# My First Git

## Project Description
Workshop 2, task 2. A simple hello world in node.js

## Installation Instructions
To install, run `git clone https://github.com/MichaelJPuccini/myFirstGit`

## Run Instructions
Either use `node index.js` or `npm start` to run the project.
